The main idea behind a non-negative quantum mechanical Bootstrap is that, using the non-negativity of the norm paired with more specialised constraints, we can reject potential values for observables. 
\begin{equation}
    \langle\mathcal{O}^{\dagger}\mathcal{O}\rangle_{\beta}\geq0\,\,,\forall\,\,\mathcal{O}\in\mathcal{H}
\end{equation}
While exact values are not always obtainable, one can find rigorous upper and lower bounds for observables.
